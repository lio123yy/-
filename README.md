# 五子棋联机对战平台

## 概念与愿景
一个具有复古未来主义风格的五子棋联机对战平台，玩家可以通过房间码与好友实时对战。融合80年代霓虹美学与现代游戏体验，带来沉浸式的竞技氛围。

## 设计语言

### 美学方向
复古未来主义 (Retro-Futurism) - 80年代霓虹风格，带有CRT扫描线效果和霓虹光晕

### 配色方案
- **主色**: #7C3AED (紫色霓虹)
- **次要色**: #A78BFA (淡紫色)
- **行动色**: #F43F5E (玫瑰红)
- **背景色**: #0F0F23 (深蓝黑)
- **文字色**: #E2E8F0 (浅灰白)

### 字体
- **标题**: Russo One
- **正文**: Chakra Petch

### 动效
- CRT扫描线叠加效果
- 霓虹光晕 (text-shadow + box-shadow)
- 故障效果 (glitch keyframes)
- 平滑过渡 (150-300ms)

## 功能

### 1. 首页/大厅
- 创建房间 (生成6位房间码)
- 加入房间 (输入房间码)
- 玩家名称设置

### 2. 游戏房间
- 15x15 标准棋盘
- 黑棋/白棋对战
- 实时同步
- 胜负判定
- 重新开始功能
- 退出房间

### 3. 房间系统
- 6位数字房间码
- 房间状态管理
- 断线重连
- 等待对手机制

## 技术方案
- **前端**: 纯 HTML/CSS/JavaScript
- **后端**: Python FastAPI + WebSocket
- **实时通信**: WebSocket

## 布局结构
1. Hero区域 - 游戏标题与标语
2. 功能区 - 创建/加入房间
3. 游戏区 - 棋盘与状态显示
